# ADR 0003: Public Branding and Canonical Links

## Status

Accepted

## Context

The repository will be public, discoverable, and associated with the personal brand and site of Ulisses Flores. Public documents and generated reports must consistently point to the canonical site and product identity.

## Decision

Adopt the following public naming rules:

- Brand: `Odisseo`
- Product: `Odisseo Signal Atlas`
- Canonical backlink target: `https://ulissesflores.com`

All public documentation and generated Markdown reports must include the canonical site link.

## Consequences

- stronger SEO, GEO, and LLM retrieval consistency
- tighter association between the open repository and the canonical site
- clearer public-facing product identity
